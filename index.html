TRTL

TRTL

TRTL is a location based social media app. Users can view and post content in your immediate area. With a user following system and intelligent post ranking, TRTL brings a location to life with content relevant to you.

Webo for Jibo

Webo for Jibo

Webo is an add on package of Skills for Jibo in order to interact with emotion. Using Jibo's built in camera functionality in conjunction with Microsoft's Emotion API, we are giving Jibo the ability to empathize, categorizing facial appearances into emotions for Jibo to interpret and react to appropriately.

MixMatch

Mixmatch

MixMatch is a mobile application that combines social networks, communities, and event management utilities to facilitate event discovery.

Alongside this solution to spontaneous event discvoery, MixMatch's value grows alongside you and your friends with its user timeline and user recognition system.

Tweather

Tweather

Tweather is a Twitter-based crowdsourced weather forecast. Tweather evaluates tweets using sentiment analysis and visualizes tweets on Google Maps to produce an intuitive app to understand the weather.
